Conhecimentos

JavaScript

JavaScript é uma linguagem de programação de alto nível usada para tornar as páginas da web interativas, permitindo adicionar funcionalidades como validação de formulários, animações e interações dinâmicas com o usuário.

HTML

HTML (HyperText Markup Language) é uma linguagem de marcação usada para criar e estruturar páginas da web, definindo elementos e organizando o conteúdo, como texto, imagens e links.

CSS

CSS (Cascading Style Sheets) é uma linguagem de estilo usada para controlar a aparência e o layout de elementos HTML em páginas da web, permitindo a definição de cores, fontes, espaçamento e posicionamento.
